COMP4321 Phase 1 - Spider + Indexer + Test Program

1) Prerequisites
- Java 21 (or newer) installed
- Maven installed (if using Maven commands)
- Internet access to crawl the test website

2) Project Structure (important files)
- src/main/java/Crawler.java
- src/main/java/Indexer.java
- src/main/java/SpiderResultExporter.java
- src/main/java/WebPageData.java
- src/main/java/CrawlerUtils/WebPageExtractor.java
- src/main/java/stopwords.txt
- database.db (generated after running crawler)
- spider_result.txt (generated by test program)

3) Build Commands

Option A: Build with Maven (recommended)
From project root:
  mvn clean compile

Option B: Build with javac directly (if Maven is unavailable)
From project root (PowerShell):
  New-Item -ItemType Directory -Force target\classes
  javac -cp "lib\jsoup-1.22.1.jar;lib\mapdb-3.1.0.jar" -d target\classes src\main\java\IRUtilities\Porter.java src\main\java\CrawlerUtils\Utils.java src\main\java\CrawlerUtils\WebPageExtractor.java src\main\java\WebPageData.java src\main\java\Indexer.java src\main\java\Crawler.java src\main\java\SpiderResultExporter.java

4) Run Spider (BFS crawl + indexing 30 pages)

Current crawler main is configured to crawl:
  https://www.cse.ust.hk/~kwtleung/COMP4321/testpage.htm
with k = 30 pages.

Run (Maven-built classes):
  java -cp "target\classes;lib\jsoup-1.22.1.jar;lib\mapdb-3.1.0.jar" Crawler

Expected result:
- database.db is created/updated
- 30 pages are crawled/indexed (BFS)

5) Run Test Program (export DB content to spider_result.txt)

Default run:
  java -cp "target\classes;lib\jsoup-1.22.1.jar;lib\mapdb-3.1.0.jar" SpiderResultExporter

Custom DB/output path:
  java -cp "target\classes;lib\jsoup-1.22.1.jar;lib\mapdb-3.1.0.jar" SpiderResultExporter database.db spider_result.txt

Expected result:
- spider_result.txt generated in project root

6) Output Format Notes (spider_result.txt)
- One page block per indexed page
- Each block contains:
  1. Page title
  2. URL
  3. Last modification date, size of page
  4. Keyword1 freq1; Keyword2 freq2; ... (max 10)
  5. Child links, one per line (max 10)
- Separator between pages:
  --------------------------------------------------
- Missing fields are printed as:
  N/A
